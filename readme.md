# TopGames – Aplicación de reserva de videojuegos

## Descripción del proyecto

TopGames es un proyecto académico ficticio cuyo objetivo es simular la estructura y documentación de una aplicación de reserva de videojuegos.

El proyecto se centra en el diseño de la interfaz, la organización del código y la gestión del proyecto mediante control de versiones, sin implementar funcionalidad real.
