# TopGames – Aplicación de reserva de videojuegos

## Descripción del proyecto

TopGames es un proyecto académico ficticio cuyo objetivo es simular la estructura y documentación de una aplicación de reserva de videojuegos.

El proyecto se centra en el diseño de la interfaz, la organización del código y la gestión del proyecto mediante control de versiones, sin implementar funcionalidad real.
## Gestión de dependencias

Durante el desarrollo del proyecto se ha creado una rama específica llamada `Dependencies` para la gestión de dependencias externas.

En dicha rama se ha añadido la dependencia Lombok al archivo `pom.xml`.
Esta librería permite reducir código repetitivo en clases Java, como constructores, getters y setters, facilitando la mantenibilidad del proyecto.

La dependencia se ha añadido únicamente a nivel de configuración, sin implementación funcional, dejando el proyecto preparado para futuras ampliaciones.
